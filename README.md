# Contacts Book CLI

A command-line contacts manager built in Python as part of my
public learning journey at [@rebooted_engineer](https://rebooted-engineer.hashnode.dev).

## What it does

- Add, search, delete, and list contacts
- Persists data between runs using JSON
- Validates phone numbers and emails with regex
- Handles all errors gracefully — never crashes on bad input

## What I learned building this

- OOP design — separating utilities, core class, and UI layers
- JSON persistence with proper error handling
- Regular expressions for real input validation
- The difference between code that works and code that's clean

## How to run it

```bash
git clone https://github.com/rebooted_engineer/contacts-book.git
cd contacts-book
python contacts.py
```

## Tech stack

Python · JSON · Regex · OOP · CLI
